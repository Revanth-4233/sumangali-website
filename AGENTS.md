# Project Instructions

## DO NOT read or process image files
This project contains PNG images in the `images/` directory. The AI model does not support image input. Never attempt to read, process, or open any `.png`, `.jpg`, `.jpeg`, `.gif`, or `.ico` files. Skip them silently.

## Site structure
- Static HTML/CSS/JS website — no framework
- Pages: index.html (homepage), men.html, women.html, kids.html, baby.html, admin.html
- Local server: `node server.js` runs on port 3000
